Cookies Policy

A) You should first understand what cookies are? Cookies can be described as below:

i) A cookie can be defined as a file that contains an identifier in the form of a string of letters and numbers, which is sent by a web server to a web browser and is stored by the browser. The identifier is then sent back to the server each time the browser requests a page from the server.
ii) Cookies may be either persistent cookies or session cookies. A persistent cookie will be stored by a web browser and will remain valid until its set expiry date or unless it is deleted by the user before the expiry date. However, a session cookie will expire at the end of the user session, when the web browser is closed.
Iii) Cookies do not usually contain any information that personally identifies a user. However, the personal information that we store about you may be linked to the information stored in, and may be obtained from cookies.
